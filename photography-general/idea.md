# Premium Photography Portfolio Website

## Objective

Build a modern, premium, luxury photography portfolio website using **only HTML, CSS and Vanilla JavaScript**.

The website should look like a high-end ThemeForest template designed for a professional photography agency.

The overall feeling should be:

- Luxury
- Elegant
- Cinematic
- Modern
- Minimal
- Dark Theme
- Premium

The design should use large photography, generous spacing, rounded cards, subtle glass effects, smooth hover animations and warm golden accents.

The page should be long (approximately 10 full sections) with alternating image and content layouts.

---

# Technology

Only use

- HTML5
- CSS3
- Vanilla JavaScript

Do NOT use

- React
- Bootstrap
- Tailwind
- Vue
- Angular
- jQuery
- GSAP
- AOS
- External UI libraries

Everything must be coded manually.
